## Technologies

- React Native
- Expo
- Typescript
- React Native Gesture Handler
- Expo Google Fonts
- Expo Vector Icons



## Running locally

Clone the project

```bash
  git clone https://github.com/felipefreitasa/ios-note-swipeable-view
```

Enter the project directory

```bash
  cd ios-note-swipeable-view
```

Install the dependencies

```bash
  yarn
```

Start the server

```bash
  npx expo start
```
